BUNDA AMI LINKTREE RESPONSIVE

Isi folder:
1. index.html

Cara pakai:
1. Extract file ZIP.
2. Buka index.html di browser untuk melihat tampilan.
3. Edit index.html memakai VS Code, Notepad, atau editor lain.
4. Ganti link placeholder berikut dengan link asli:
   - #ganti-link-whatsapp
   - #ganti-link-landing-page
   - #ganti-link-zoom
   - #ganti-link-facebook
   - #ganti-link-instagram
   - #ganti-link-tiktok

Cara ganti foto profil:
Cari bagian ini:
<div class="avatar">BA</div>

Lalu ganti menjadi contoh seperti ini:
<img class="avatar" src="foto-bunda-ami.jpg" alt="Foto Bunda Ami" />

Pastikan file foto berada satu folder dengan index.html, atau sesuaikan nama folder/path gambarnya.

Responsive:
- HP: tombol tampil 1 kolom, teks dibuat lebih ringkas.
- Tablet: tombol tampil lebih lega, beberapa link menjadi 2 kolom.
- Desktop: tetap rapi dan tidak melebar berlebihan.
